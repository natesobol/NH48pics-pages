NH48 API Web App

Peak catalog • Detail pages • Photos

Explore New Hampshire’s 48 four-thousand-footers with a searchable catalog UI and per-peak detail pages. The app reads directly from the NH48 API to display each peak’s elevation, prominence, range, difficulty, standard routes, and rich photo metadata—perfect for trip-planning or as a live demo of the API.

WMNF Trails API Web App

Trail explorer • Progress tracker • 3-D hike

Browse White Mountain trails with a searchable list and interactive map powered by the WMNF Trails API. Mark trails as completed, record dates, hide completed routes, export or import your progress, and launch an immersive 3-D “Virtual Hike” over the trail in Cesium. Not for navigation—always verify against official maps and guidebooks.

Developer APIs

Open-source datasets and code for hikers, developers, educators, and conservationists.

NH48 API

A structured data and media system for all 48 New Hampshire four-thousand-foot peaks. Includes peak attributes, trail and route metadata, and EXIF-driven photo manifests. Distributed as static JSON and images via GitHub Pages and jsDelivr for easy reuse in apps and maps.

View on GitHub

WMNF Trails API

A GeoJSON trail dataset and front-end tools for the White Mountain National Forest. Includes a Trail Explorer with search/sort controls, completion tracking with date fields and CSV/XLSX import/export, plus a Cesium-powered Virtual Hike viewer for individual trails.
